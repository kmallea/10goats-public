# 🐐 10GOATS Public

Welcome to the official public repo for **10GOATS**.

This is where the community helps make the platform better.

If you find a bug, something feels off, or you’ve got an idea — this is the spot.

---

## 🔥 What is this repo for?

- 🐛 Report bugs
- 💡 Suggest features
- 🧠 Share ideas to improve the experience

You don’t need to be technical. If something feels wrong, that’s enough.

---

## 🐛 Report a Bug

Click the **"Issues"** tab and choose **Bug Report**.

Try to include:
- What happened
- What you expected
- Steps to reproduce (if you can)
- Screenshot (super helpful)

Even small stuff matters.

---

## 💡 Suggest a Feature

Got an idea that would make 10GOATS better?

Open an issue and choose **Feature Request**.

This app is built around community opinions — that includes how it evolves.

Not algorithms. Not critics. **The people.**
